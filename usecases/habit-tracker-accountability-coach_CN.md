# 习惯追踪器与问责教练

你尝试过所有的习惯追踪应用。它们都管用一周，然后你就不再打开它们了。问题不在于应用——而在于追踪习惯是被动的。如果你的智能体主动联系你，问你今天过得怎么样，并根据你是在连续坚持还是正在懈怠来调整它的方法呢？

这个用例将 OpenClaw 转变为一个主动的问责伙伴，通过 Telegram 或 SMS 每天与你联系。

## 痛点

习惯应用依赖你记得打开它们。推送通知很容易被忽略。对行为改变真正有效的是**主动问责**——某人（或某物）直接问你、庆祝你的胜利、在你懈怠时轻推你。这个智能体正是这样做的，而且不会像打扰朋友那样尴尬。

## 它做什么

- **每日签到**，通过 Telegram 或 SMS 在你选择的时间（例如，早上7点用于晨间例程，晚上9点用于一天结束回顾）
- **追踪你定义的习惯**——运动、阅读、冥想、饮水量、编程，任何对你重要的事
- **连续记录追踪**——知道你每个习惯的当前连续天数，并在消息中引用它
- **自适应轻推**——根据你的表现调整语气（当你保持一致时鼓励，当你错过几天时温和坚持）
- **每周报告**——总结你的一周，包括完成率、最长连续天数和模式（例如，"你倾向于在周三跳过锻炼"）

## 你需要的技能

- Telegram 或 SMS 集成（Twilio 用于 SMS，或 Telegram Bot API）
- 调度 / cron 用于定时签到
- 文件系统或数据库访问用于存储习惯数据
- 可选：Google Sheets 集成用于可视化习惯仪表板

## 如何设置

1. 定义你的习惯和签到时间表：
```text
我想让你做我的问责教练。为我追踪这些日常习惯：

1. 晨练（早上7:30签到）
2. 阅读30分钟（晚上8:00签到）
3. 中午前不看社交媒体（中午12:30签到）
4. 喝8杯水（下午6:00签到）

在每个签到时间给我发 Telegram 消息，问我是否完成了习惯。在本地文件中记录我的连续天数。
```

2. 设置追踪和语气：
```text
当我确认一个习惯时，用简短的鼓励消息回复并提及我当前的连续天数。例如："晨练第12天。很棒。"

当我错过一个习惯时，不要让我内疚。只需承认它并提醒我为什么开始。如果我连续错过3天，发送更长的激励消息并问我是否要调整目标。

如果我2小时内没有回复签到，发送一次跟进。之后不要再发消息骚扰我。
```

3. 添加每周报告：
```text
每周日上午10点，给我发送每周总结：
- 每个习惯的完成率
- 当前连续天数
- 最好的一天和最差的一天
- 你注意到的一个模式（例如，"你总是在周五跳过阅读"）
- 下周的一个建议

将所有数据存储在 ~/habits/log.json 中，这样我可以随时查看历史记录。
```

4. 可选——通过 Google Sheets 的可视化仪表板：
```text
每天结束时，用今天的习惯数据更新 Google Sheet。
列：日期、锻炼、阅读、不看社交媒体、饮水、备注。
用 ✓ 标记完成的习慣，用 ✗ 标记错过的。
```

## 关键洞察

- **自适应语气**是它与 cron 作业不同的地方。静态提醒会被忽略。说"第15天了，现在不要打破它"的消息实际上能激励你。
- 保持追踪的习惯数量较少（3-5个）。追踪太多会导致签到疲劳，你会开始忽略消息。
- 每周模式分析出奇地有用——你会发现诸如"我在有早会的日子从不锻炼"之类的事情，并可以围绕它做计划。
- 如果你想将习惯与感觉关联，可以与[健康与症状追踪器](health-symptom-tracker.md)配对使用。
