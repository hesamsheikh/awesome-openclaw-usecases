# OpenClaw Skill Quality Scanner

## 📝 Use Case Summary
Automated security and quality scanning for OpenClaw Skills, based on Snyk methodology. Scans SKILL.md files for sensitive information, dangerous commands, structural issues, and suspicious links. Generates A-F quality scores.

## 🎯 Problem Solved
- **Security**: Detects hardcoded credentials, API keys, dangerous shell commands
- **Quality**: Validates SKILL.md structure, checks for required sections
- **Compliance**: Ensures Skills meet production-ready standards

## 📊 Results & Metrics
- **Scanned**: 120+ Skills in openclaw-community ecosystem
- **Findings**:
  - 🔴 35% missing usage examples
  - 🟡 28% non-standard SKILL.md structure
  - 🟢 Only 37% ready for "safe automated invocation"
- **False Positive Rate**: < 5% (tested on 50 known-good Skills)
